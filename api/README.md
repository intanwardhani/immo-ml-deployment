# API backend for Immo Price Predictor
This directory contains the API code for the price prediction app.

# Files

- `__init__.py`     : Initialise package
- `app.py`          : Main functions running the API
- `config.py`       : Configuration of global variables
- Dockerfile
- `model_loader.py` : Function to load prediction model
- `predict.py`      : Actual prediction function
- `schemas.py`      : Pydantic input validator
